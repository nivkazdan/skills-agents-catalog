## Summary

**Trigger Phrases:** `create retell agent`, `retell voice agent`, `configure retell agent`, `retell agent setup`

### What This Skill Does

This skill guides developers through creating fully-configured Retell AI voice agents. It covers setting up custom system prompts for agent behavior, selecting appropriate voices from the voice library, configuring response timing and interruption handling, and setting up agent-specific LLM parameters like temperature and max tokens.

### Technical Success Criteria

- Working voice agent with custom system prompt
- Voice selected and configured from Retell voice library
- Behavior parameters set (interruption handling, response delay)
- Agent tested with sample conversation

### Business Success Criteria

- Reduced time-to-first-agent for new projects
- Standardized agent configurations across team
- Deploy first functional voice agent within 30 minutes
