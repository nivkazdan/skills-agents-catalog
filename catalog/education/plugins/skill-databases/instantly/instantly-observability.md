# instantly-observability

> Implement monitoring, alerting, and logging for comprehensive Instantly platform visibility

## Directory Structure

```
instantly-observability/

└── examples/

    ├── datadog_dashboard.json
```

## File Descriptions

| File | Type | Purpose |
|------|------|---------|
| SKILL.md | Markdown | Observability architecture and implementation guide |
| metrics_exporter.py | Python | Export Instantly metrics to monitoring systems |
| datadog_dashboard.json | JSON | Pre-built Datadog dashboard for Instantly |
| prometheus_config.yml | YAML | Prometheus configuration for metrics collection |
| alert_rules.yml | YAML | Alert rules for campaign and delivery issues |
| logging_config.py | Python | Structured logging configuration |
