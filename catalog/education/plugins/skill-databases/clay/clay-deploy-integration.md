## File Descriptions

### 1. SKILL.md
**Purpose:** Deploy Clay integrations to serverless platforms including AWS Lambda, Google Cloud Functions, and Vercel. Covers secrets management and production configuration.
**Workflow:** Deployment skill. Use when deploying Clay-powered applications to production.
**Relates to:** Follows clay-ci-integration for pipeline setup; integrates with clay-prod-checklist for validation.

## Summary

This skill guides deployment of Clay integrations to major cloud platforms. It covers configuring AWS Lambda with Clay API access, deploying to Google Cloud Functions, setting up Vercel serverless functions, implementing secure secrets management for each platform, configuring health check endpoints, and setting up monitoring integrations. Enables flexible infrastructure choices.
