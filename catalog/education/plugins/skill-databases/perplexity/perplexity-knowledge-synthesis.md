# perplexity-knowledge-synthesis

> Synthesize knowledge from multiple Perplexity searches

## Directory Structure

```
perplexity-knowledge-synthesis/
├── SKILL.md                    # Main skill definition with YAML frontmatter
└── examples/                   # Optional examples directory
    ├── synthesizer.py          # Knowledge synthesis engine
    ├── fact_extractor.py       # Extract facts from responses
    ├── knowledge_graph.py      # Build knowledge graphs
    └── synthesis_report.py     # Generate synthesis reports
```

## File Descriptions

| File | Type | Purpose |
|------|------|---------|
| `SKILL.md` | Markdown | Skill definition with synthesis patterns |
| `synthesizer.py` | Python | Synthesize knowledge from multiple sources |
| `fact_extractor.py` | Python | Extract key facts and claims |
| `knowledge_graph.py` | Python | Build knowledge graph from facts |
| `synthesis_report.py` | Python | Generate comprehensive reports |

## Summary

**Category:** advanced
**Target Audience:** Developer building knowledge systems
**Trigger Phrases:** `perplexity synthesis`, `aggregate knowledge`, `combine research`, `knowledge base`

### What This Skill Does

This skill teaches knowledge synthesis from searches:

- Aggregating information across searches
- Extracting facts and claims
- Building knowledge graphs
- Identifying conflicts and consensus
- Generating synthesis reports

### Technical Success Criteria

- Coherent knowledge synthesis from multiple sources
- Facts properly extracted and linked
- Conflicts identified and documented

### Business Success Criteria

- Comprehensive understanding from search
- Reduced manual synthesis work
- Actionable knowledge outputs
