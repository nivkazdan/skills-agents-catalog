### 1. SKILL.md
**Purpose:** Configure team-based access control, API key management, and usage quotas for enterprise FireCrawl deployments.
**Workflow:** Enterprise governance skill - establishes access control and usage governance for team environments.
**Relates to:** Extends firecrawl-multi-env-setup; complements firecrawl-data-handling for compliance

## Summary

This skill implements enterprise access control for FireCrawl operations. It covers team-based API key management, role-based access control (admin, developer, viewer), per-team usage quotas and rate limits, API key rotation procedures, usage tracking and reporting, and audit logging for compliance. These controls enable safe, governed use of FireCrawl across enterprise teams with proper accountability.
