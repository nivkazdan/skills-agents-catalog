## File Descriptions

### 1. SKILL.md
**Purpose:** Apply advanced debugging for hard-to-diagnose Exa issues including relevance problems, edge cases, and intermittent failures.
**Workflow:** Escalation skill when standard troubleshooting fails - for complex issues.
**Relates to:** Follows exa-debug-bundle; provides input for support escalation

## Summary

This skill covers advanced Exa troubleshooting: relevance analysis and query optimization, edge case identification (unusual domains, languages, content types), timing analysis for intermittent failures, network trace analysis for connectivity issues, search mode comparison testing, minimal reproduction creation, A/B testing for query variations, support escalation with comprehensive evidence, and post-resolution documentation. Target: Successfully diagnose 95% of escalated issues.
