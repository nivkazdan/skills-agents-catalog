## File Descriptions

### 1. SKILL.md
**Purpose:** Implement similarity search using Exa's find_similar endpoint for content discovery, recommendations, and related content features.
**Workflow:** Content discovery skill - enables "more like this" functionality.
**Relates to:** Follows exa-filters-domains; complementary to exa-embedding-integration

## Summary

This skill covers Exa's find_similar endpoint: finding content similar to a given URL, configuring similarity thresholds, combining similarity with domain filters, deduplication strategies, building recommendation systems, content clustering patterns, and recursive similarity exploration (finding similar to similar). Includes use cases for content discovery, competitive analysis, and research workflows.
