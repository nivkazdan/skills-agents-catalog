## Summary

This skill provides enterprise-grade access control for Supabase integrations. It covers role definitions with permissions matrix, SAML and OAuth2/OIDC SSO integration, organization management, access control middleware for Express, and audit trail implementation. Use this skill when implementing SSO integration, configuring role-based permissions, or setting up enterprise security controls.
