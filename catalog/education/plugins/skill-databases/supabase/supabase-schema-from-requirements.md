**Purpose:** Execute Supabase primary workflow: generate database schema from natural language requirements. This is the main workflow for starting new Supabase projects.

## Summary

This skill guides the process of generating a Supabase database schema from natural language requirements. It covers requirement analysis, schema design, migration file creation, and deployment. Use this skill when starting a new Supabase project with defined data requirements, refactoring an existing schema for new features, or creating migrations from specification documents.
