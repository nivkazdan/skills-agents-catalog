## File Descriptions

### 1. SKILL.md
**Purpose:** Provides a comprehensive reference for all Cursor keyboard shortcuts including AI-specific commands, editor operations, and customization options.

**Workflow:** This skill activates when users want to learn or customize keyboard shortcuts. It covers AI commands (Chat, Composer, Inline Edit, completions), standard editor shortcuts, multi-cursor operations, and Vim mode integration.

**Relates to:** cursor-local-dev-loop (workflow efficiency), cursor-ai-chat (chat shortcuts), cursor-tab-completion (completion controls), cursor-extension-integration (extension keybindings)

## Summary

This reference skill provides complete keyboard shortcut documentation for Cursor on macOS, Windows, and Linux. It covers AI-specific shortcuts (Cmd+L for chat, Cmd+I for Composer, Cmd+K for inline edit), completion controls (Tab, Esc, partial acceptance), standard editor operations, multi-cursor editing, and customization through keybindings.json. The skill includes a quick reference card and Vim mode configuration.
