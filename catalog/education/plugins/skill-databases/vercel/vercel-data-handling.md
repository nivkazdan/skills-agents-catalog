## File Descriptions

### 1. SKILL.md
**Purpose:** Implement PII detection, data redaction, retention policies, and GDPR/CCPA compliance patterns.
**Workflow:** Applied during integration development; audited during compliance reviews.
**Relates to:** Follows vercel-incident-runbook; leads to vercel-enterprise-rbac for access control

## Summary

This skill addresses data privacy and compliance for Vercel integrations. It covers data classification by sensitivity (PII, Sensitive, Business, Public), regex-based PII detection for emails, phones, SSNs, and credit cards, redaction functions for logging, retention period configuration with automatic cleanup jobs, GDPR Data Subject Access Request implementation, right to deletion handlers, and data minimization patterns. The goal is 100% compliance with privacy regulations.
