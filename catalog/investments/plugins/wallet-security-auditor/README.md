# Wallet Security Auditor

Audits crypto wallet implementations for common security issues (key management, signing flows, transaction construction, and related vulnerability patterns).

## Install

```bash
/plugin install wallet-security-auditor
```

## Use

- Invoke the skill manually: `/auditing-wallet-security`
- Or describe the wallet/security auditing task and Claude Code will auto-activate the skill when relevant.

## Included

- Skill: `auditing-wallet-security` (`plugins/crypto/wallet-security-auditor/skills/auditing-wallet-security/SKILL.md`)
