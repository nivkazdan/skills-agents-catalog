# Scripts

Bundled resources for regression-analysis-tool skill

- [x] validate_data.py: Script to validate the input dataset for regression analysis, checking for missing values, outliers, and data type consistency.
- [x] generate_model.py: Script to automatically generate and train regression models based on the input data and specified parameters.
- [x] evaluate_model.py: Script to evaluate the performance of the trained regression model using metrics like R-squared, Mean Squared Error (MSE), etc.
