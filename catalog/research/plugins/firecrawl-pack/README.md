# FireCrawl Skill Pack

## Installation

```bash
/plugin install firecrawl-pack@claude-code-plugins-plus
```

## Usage

Skills trigger automatically when you discuss FireCrawl topics. For example:

- "Help me set up FireCrawl" → triggers `firecrawl-install-auth`
- "Debug this FireCrawl error" → triggers `firecrawl-common-errors`
- "Deploy my FireCrawl integration" → triggers `firecrawl-deploy-integration`
