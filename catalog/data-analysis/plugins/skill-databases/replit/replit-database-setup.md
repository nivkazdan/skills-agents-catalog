## File Descriptions

### 1. SKILL.md
**Purpose:** Configure Replit Database (key-value store) or connect external databases like PostgreSQL and MongoDB.
**Workflow:** Data persistence skill - enables stateful applications beyond simple scripts.
**Relates to:** Builds on replit-secrets-management; foundation for replit-agent-advanced and production apps.

## Summary

This skill covers data persistence options on Replit. It teaches developers to use Replit's built-in key-value Database for simple storage needs, connect PostgreSQL databases for relational data, integrate MongoDB for document storage, configure connection strings securely via Secrets, and implement basic CRUD operations. This enables building real applications that persist data across sessions and deployments.
