# posthog-observability

> Implement comprehensive observability for PostHog including metrics logging and alerting

## Directory Structure

```
posthog-observability/
├── SKILL.md
└── examples/
    ├── prometheus_config.yml
    ├── grafana_dashboard.json
    ├── alerting_rules.yml
    └── log_aggregation.md
```

## File Descriptions

| File | Type | Purpose |
|------|------|---------|
| SKILL.md | Markdown | Core skill instructions for PostHog observability implementation |
| prometheus_config.yml | YAML | Prometheus configuration for PostHog metrics scraping |
| grafana_dashboard.json | JSON | Pre-built Grafana dashboard for PostHog health monitoring |
| alerting_rules.yml | YAML | Alert definitions for PostHog failure modes |
| log_aggregation.md | Markdown | Guide for centralizing PostHog logs |
