## Summary

**Target Audience:** New Retell AI users, Developers learning voice AI
**Trigger Phrases:** `retell hello world`, `retell example`, `retell quick start`, `simple retell voice agent`

### What This Skill Does

This skill creates a minimal working Retell AI example that demonstrates the core API patterns for voice agent creation. It shows developers how to initialize the Retell client, create a basic voice agent with a simple prompt, and test the agent configuration. This serves as the foundation for understanding how Retell AI voice agents work.

### Technical Success Criteria

- Working code file with Retell client initialization
- Successfully created voice agent via API
- Agent configured with basic prompt and default voice

### Business Success Criteria

- Faster developer onboarding to voice AI concepts
- Reduced learning curve for Retell AI platform
- Execute first successful Retell AI API call within minutes of setup
