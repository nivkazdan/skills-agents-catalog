## File Descriptions

### 1. SKILL.md
**Purpose:** Implement PII detection, content filtering, and data retention policies for scraped content in compliance with regulations.
**Workflow:** Compliance skill - ensures scraped data is handled according to privacy regulations and ethical standards.
**Relates to:** Extends firecrawl-security-basics; supports firecrawl-enterprise-rbac for access control

## Summary

This skill implements responsible data handling for scraped content. It covers PII detection and redaction (emails, phone numbers, addresses, names), content classification, retention policy enforcement, GDPR/CCPA compliance patterns, data export and deletion capabilities, and audit trail maintenance. These practices ensure scraped data is handled ethically and in compliance with privacy regulations.
