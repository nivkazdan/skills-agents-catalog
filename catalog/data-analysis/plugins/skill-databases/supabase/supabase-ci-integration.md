### 1. SKILL.md
**Purpose:** Configure Supabase CI/CD integration with GitHub Actions including automated testing, secret management, and release workflows.
**Workflow:** First CI/CD skill. Establishes automated testing and deployment foundation.
**Relates to:** Enables supabase-deploy-integration; supports supabase-upgrade-migration with automated testing.

## Summary

This skill provides guidance on setting up CI/CD pipelines for Supabase integrations. It covers creating GitHub Actions workflows, configuring secrets with gh CLI, writing integration tests, setting up release workflows, and configuring branch protection. Use this skill when setting up automated testing for a Supabase project or when integrating Supabase tests into an existing CI/CD pipeline.
