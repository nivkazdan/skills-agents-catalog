## Summary

**Category:** onboarding
**Target Audience:** Developer new to OpenRouter
**Trigger Phrases:** `openrouter hello world`, `openrouter first request`, `openrouter quickstart`, `test openrouter`

### What This Skill Does

This skill provides a quick-start guide for making your first API request with OpenRouter. It covers:

- Minimal cURL and Python examples
- Understanding response structure
- Model selection basics
- Token usage tracking
- Verifying successful integration

### Technical Success Criteria

- Successfully make first API call and receive response
- Understanding of request/response pattern
- Token usage visible in response

### Business Success Criteria

- Quick validation of API integration readiness
- Developer confidence in OpenRouter capabilities
- Foundation for more complex integrations
