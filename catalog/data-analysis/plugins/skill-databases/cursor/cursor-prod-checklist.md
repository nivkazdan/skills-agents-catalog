### 1. SKILL.md
**Purpose:** Provides a comprehensive production readiness checklist for Cursor IDE setup covering authentication, configuration, security, team settings, and maintenance schedules.

**Workflow:** This skill activates when preparing Cursor for team deployment or production use. It provides checklists for initial setup, project configuration, performance optimization, security, workflow integration, and ongoing maintenance.

**Relates to:** cursor-team-setup (team deployment), cursor-rules-config (.cursorrules setup), cursor-privacy-settings (security configuration), cursor-reference-architecture (architecture patterns)

## Summary

This operational skill provides a complete production readiness checklist for Cursor IDE deployments. It covers authentication and licensing verification, core configuration (models, privacy mode), project configuration (.cursorrules, indexing), performance optimization, security settings, team configuration (shared settings, onboarding), workflow integration (git, CI/CD), backup/recovery, verification tests, and maintenance schedules (weekly, monthly, quarterly).
