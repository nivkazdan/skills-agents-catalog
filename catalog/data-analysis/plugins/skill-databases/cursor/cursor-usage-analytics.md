## File Descriptions

### 1. SKILL.md
**Purpose:** Tracks and analyzes Cursor usage metrics including user activity, AI feature adoption, costs, and team performance through the admin dashboard and API.

**Workflow:** This skill activates when team admins or managers need usage insights. It covers available metrics, dashboard views, custom reports, analytics API, benchmarking, optimization insights, and privacy compliance.

**Relates to:** cursor-team-setup (team management), cursor-api-key-management (cost tracking), cursor-compliance-audit (usage audit), cursor-model-selection (model usage analysis)

## Summary

This analytics skill covers comprehensive usage tracking for Cursor teams. It includes available metrics (user activity, AI usage, costs), key metrics explained (completion acceptance rate, chat sessions, composer usage), dashboard views (executive summary, team breakdown, individual user), custom and scheduled reports, analytics API with endpoints and integration examples, industry benchmarking (typical acceptance rates, feature usage), optimization insights (low acceptance, underutilization, high costs), privacy compliance (what data is/isn't tracked), and best practices for regular review.
