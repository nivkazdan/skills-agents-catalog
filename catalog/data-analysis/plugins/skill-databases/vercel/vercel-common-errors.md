## File Descriptions

### 1. SKILL.md
**Purpose:** Diagnose and fix top 10 most common Vercel errors including build failures, function timeouts, and domain verification issues.
**Workflow:** On-demand troubleshooting - consult when encountering errors during development or production.
**Relates to:** Entry point to vercel-debug-bundle for complex issues; integrates with vercel-incident-runbook for production outages

## Summary

This skill serves as a quick reference for the most frequently encountered Vercel errors. It covers build failures, function invocation timeouts, domain verification issues, and authentication errors. Each error includes the error message, root cause explanation, and step-by-step solution. The escalation path guides developers to more advanced debugging when these common fixes don't resolve the issue.
