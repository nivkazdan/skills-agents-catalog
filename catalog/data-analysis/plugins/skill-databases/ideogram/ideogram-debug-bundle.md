# ideogram-debug-bundle

> Collect debug information for Ideogram support tickets and comprehensive issue diagnosis.

```

```

## File Descriptions

| File | Type | Purpose |
|------|------|---------|
| SKILL.md | Markdown | Debug bundle collection procedure with required information and format |
| examples/collect_bundle.py | Python | Script to automatically collect comprehensive debug information |

## Summary

**Category:** Advanced
**Target Audience:** DevOps engineers, Support engineers, SREs
**Trigger Phrases:** ideogram debug info, ideogram support ticket, gather ideogram diagnostics, ideogram debug bundle

This skill collects comprehensive debug information for Ideogram support tickets. It gathers API version, request/response logs, configuration snapshot, error details, environment information, and connectivity test results. Use this skill when preparing support tickets or documenting issues for escalation.
