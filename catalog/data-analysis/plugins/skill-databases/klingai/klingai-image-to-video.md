## Summary

**Category:** advanced
**Target Audience:** Content creator or developer
**Trigger Phrases:** `klingai image to video`, `kling ai animate image`, `klingai image animation`, `motion from image`

### What This Skill Does

This skill teaches generating videos from static images with Kling AI. It covers:

- Image-to-video API endpoint usage
- Image preparation and requirements
- Motion description in prompts
- Animation style control
- Duration and resolution options
- Combining with text guidance

### Technical Success Criteria

- Animated video from static image with controlled motion
- Smooth and natural animation
- Motion matching prompt intent

### Business Success Criteria

- Creative content from existing image assets
- Extended use of existing visual assets
- New content formats from still imagery
