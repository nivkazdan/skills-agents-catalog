### What This Skill Does

This skill teaches optimizing Kling AI for better performance. It covers:

- Performance benchmarking and profiling
- Speed vs quality vs cost tradeoffs
- Model selection for performance
- Duration and resolution optimization
- Caching strategies for repeated generations
- Configuration comparison and recommendations

### Technical Success Criteria

- Measurable improvement in generation performance
- Optimal configuration for use case
- Caching reducing redundant generations

### Business Success Criteria

- Cost-effective high-quality video production
- Predictable generation times
- Efficient resource utilization
