## Summary

**Category:** enterprise
**Target Audience:** Solutions architect
**Trigger Phrases:** `klingai architecture`, `kling ai system design`, `klingai microservices`, `scale klingai`

### What This Skill Does

This skill provides production-ready reference architecture for Kling AI platforms. It covers:

- Microservices architecture patterns
- API Gateway and load balancing
- Queue-based video processing
- Storage tier design
- Caching strategies
- Monitoring and observability stack
- Disaster recovery and high availability

### Technical Success Criteria

- Complete architecture with microservices design
- Scalability patterns implemented
- High availability configuration

### Business Success Criteria

- Scalable and maintainable video platform
- Clear architectural decision records
- Team alignment on system design
