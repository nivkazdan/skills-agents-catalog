# instantly-install-auth

> Configure API authentication and install Instantly SDK for secure programmatic access to cold email platform

## Directory Structure

```
instantly-install-auth/
├── SKILL.md
└── examples/
    ├── auth_setup.py
    ├── auth_setup.ts
    └── test_connection.sh
```

## File Descriptions

| File | Type | Purpose |
|------|------|---------|
| SKILL.md | Markdown | Complete authentication setup instructions with API key configuration |
| auth_setup.py | Python | Python SDK installation and authentication example |
| auth_setup.ts | TypeScript | Node.js/TypeScript authentication configuration |
| test_connection.sh | Shell | Validate API connection and credentials |
