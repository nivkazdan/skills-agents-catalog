# vastai-common-errors

> Diagnose and resolve frequent Vast.ai errors including SSH failures, OOM, bid rejections, and CUDA issues

## Directory Structure

```
vastai-common-errors/
├── SKILL.md
└── examples/
    └── error_diagnostics.sh
```

## File Descriptions

| File | Type | Purpose |
|------|------|---------|
| SKILL.md | Markdown | Comprehensive error catalog with symptoms, causes, and resolution steps |
| examples/error_diagnostics.sh | Shell | Diagnostic script that checks common failure modes and suggests fixes |
