### 1. SKILL.md
**Purpose:** Best practices for using Sentry SDK in TypeScript and Python including centralized error handlers, async wrappers, and scoped context patterns.

## Summary

This skill provides best practices and patterns for using the Sentry SDK effectively in production applications. It covers centralized error handlers, async error wrappers, Express middleware patterns, scoped context usage, and breadcrumb management. Use this skill when designing error handling architecture or reviewing existing Sentry code for improvements.
