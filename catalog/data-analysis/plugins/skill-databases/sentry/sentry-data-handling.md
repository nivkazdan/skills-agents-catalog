## File Descriptions

### 1. SKILL.md
**Purpose:** Handle sensitive data properly in Sentry with PII scrubbing, GDPR compliance, data retention configuration, and IP address anonymization.
**Workflow:** Configure during security review or compliance preparation. Ensures sensitive data is properly handled and compliance requirements are met.
**Relates to:** Extends `sentry-security-basics`. Works with `sentry-policy-guardrails` for organizational standards.

## Summary

This skill configures Sentry to handle sensitive data properly and maintain compliance. It covers built-in and custom PII scrubbing, server-side data scrubbing rules, GDPR compliance including right to erasure, user consent handling, data retention configuration, IP address handling, and compliance checklists for GDPR, HIPAA, and PCI-DSS. Use this skill for regulatory compliance.
