## Usage

Skills trigger automatically when you discuss Vast.ai topics. For example:

- "Help me set up Vast.ai" → triggers `vastai-install-auth`
- "Debug this Vast.ai error" → triggers `vastai-common-errors`
- "Deploy my Vast.ai integration" → triggers `vastai-deploy-integration`
