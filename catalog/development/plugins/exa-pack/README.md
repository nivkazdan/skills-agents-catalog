## Usage

Skills trigger automatically when you discuss Exa topics. For example:

- "Help me set up Exa" → triggers `exa-install-auth`
- "Debug this Exa error" → triggers `exa-common-errors`
- "Deploy my Exa integration" → triggers `exa-deploy-integration`
