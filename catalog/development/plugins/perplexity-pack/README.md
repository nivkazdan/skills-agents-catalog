## Usage

Skills trigger automatically when you discuss Perplexity topics. For example:

- "Help me set up Perplexity" → triggers `perplexity-install-auth`
- "Debug this Perplexity error" → triggers `perplexity-common-errors`
- "Deploy my Perplexity integration" → triggers `perplexity-deploy-integration`
