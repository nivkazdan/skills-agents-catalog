### 1. SKILL.md
**Purpose:** Use advanced Replit Agent features including multi-step projects, database integration, and deployment automation.

## Summary

This skill covers advanced Replit Agent capabilities for power users. It teaches guiding Agent through multi-component full-stack applications, integrating databases and external APIs via Agent prompts, debugging and iterating on Agent-generated code effectively, combining Agent with manual coding for hybrid development, using Agent for refactoring and code improvement, automating deployment through Agent conversations, creating complex UIs with Agent assistance, handling Agent limitations and knowing when to code manually, and building production-ready applications entirely through AI collaboration.
