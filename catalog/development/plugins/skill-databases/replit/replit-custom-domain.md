## File Descriptions

### 1. SKILL.md
**Purpose:** Configure custom domains for Replit deployments with SSL certificates and DNS management.
**Workflow:** Branding and production skill - replaces default .replit.dev URLs with professional custom domains.
**Relates to:** Follows replit-deployments; enhances replit-always-on-config for production.

## Summary

This skill configures custom domains for Replit applications. It covers purchasing or transferring domain names, configuring DNS records (CNAME, A records) at registrars, adding custom domains in Replit deployment settings, automatic SSL certificate provisioning via Let's Encrypt, troubleshooting DNS propagation issues, setting up subdomain routing for multiple apps, and configuring apex domains vs www subdomains. Professional domains improve user trust and brand recognition.
