## File Descriptions

### 1. SKILL.md
**Purpose:** Configure GitHub integration for two-way sync, PR previews, and automated deployments from repositories.
**Workflow:** Version control skill - connects Replit to professional Git workflows.
**Relates to:** Foundation for replit-testing-workflow; enables team collaboration via replit-team-management.

## Summary

This skill configures GitHub integration for Replit projects. It covers connecting GitHub accounts to Replit, importing existing repositories as Repls, enabling bidirectional Git sync (push/pull), setting up automatic PR preview deployments, configuring branch-based deployment strategies, managing merge conflicts between Replit and GitHub, and using Git commands within the Replit shell. GitHub integration brings professional version control to Replit's cloud IDE experience.
