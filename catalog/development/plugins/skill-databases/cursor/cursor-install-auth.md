## File Descriptions

### 1. SKILL.md
**Purpose:** Provides comprehensive guidance for installing Cursor IDE on macOS, Linux, and Windows, and configuring authentication methods including GitHub, Google, and email sign-in.

**Workflow:** This skill activates when users need to install Cursor or set up authentication. It covers installation methods (Homebrew, AppImage, winget), authentication flows, license tiers (Free/Pro/Business), and API key configuration for using custom AI providers.

**Relates to:** cursor-hello-world (next step after installation), cursor-api-key-management (advanced API key setup), cursor-team-setup (team authentication)

## Summary

This foundational onboarding skill guides new users through the complete Cursor IDE installation and authentication process. It covers platform-specific installation methods, multiple sign-in options (GitHub, Google, Email), license tier comparison, API key configuration for custom providers, and a verification checklist to ensure successful setup. The skill also includes troubleshooting for common authentication issues.
