## File Descriptions

### 1. SKILL.md
**Purpose:** Guides first-time users through creating their first AI-powered project in Cursor, demonstrating core features like tab completion, AI chat, and Composer.

**Workflow:** This skill activates when new users want to explore Cursor's AI capabilities. It provides a 5-minute quick start, hands-on exercises for each feature, and explains the AI context system including model selection and privacy modes.

**Relates to:** cursor-install-auth (prerequisite), cursor-tab-completion (deep dive into completions), cursor-ai-chat (deep dive into chat), cursor-composer-workflows (deep dive into Composer)

## Summary

This beginner-friendly skill provides a guided first experience with Cursor's AI features through practical exercises. Users learn tab completion by writing a fibonacci function, explore AI chat for code explanations and refactoring, and use Composer for multi-file generation. The skill covers essential concepts like context windows, model selection, and privacy modes to build a solid foundation.
