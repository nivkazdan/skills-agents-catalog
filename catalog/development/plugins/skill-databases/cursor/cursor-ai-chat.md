## File Descriptions

### 1. SKILL.md
**Purpose:** Masters the Cursor AI chat interface for code assistance, covering effective chat patterns, context management with @-mentions, and model selection strategies.

**Workflow:** This skill activates when developers use chat for code explanation, generation, refactoring, or debugging. It provides prompt patterns, context management techniques, chat commands, and guidance on multi-turn conversations.

**Relates to:** cursor-context-management (context optimization), cursor-model-selection (choosing models), cursor-custom-prompts (prompt engineering), cursor-composer-workflows (when to use Composer instead)

## Summary

This core skill teaches mastery of Cursor's AI chat interface for code assistance. It covers effective chat patterns for explanation, generation, refactoring, and debugging, plus context management using @-mentions (@filename, @codebase, @docs, @web). The skill explains model selection (GPT-4, Claude), chat commands (/edit, /explain, /fix), multi-turn conversation strategies, and tips for getting better AI responses.
