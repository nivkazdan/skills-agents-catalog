## File Descriptions

### 1. SKILL.md
**Purpose:** Guides upgrading Cursor versions and migrating settings between machines or from VS Code, including backup procedures and rollback strategies.

**Workflow:** This skill activates when users need to upgrade Cursor or migrate configurations. It covers version checking, upgrade methods, pre-upgrade backup, migration between machines, VS Code migration, and troubleshooting.

**Relates to:** cursor-install-auth (fresh installation), cursor-extension-integration (extension migration), cursor-team-setup (team coordination for upgrades)

## Summary

This maintenance skill covers complete procedures for Cursor upgrades and migrations. It includes checking current version, auto-update vs manual update methods for all platforms, pre-upgrade backup checklists (settings, extensions, keybindings), Settings Sync usage, manual migration between machines, VS Code to Cursor migration (automatic import and manual), post-upgrade verification, troubleshooting (settings not preserved, extensions missing, performance issues), and rollback procedures.
