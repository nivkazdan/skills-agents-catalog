## File Descriptions

### 1. SKILL.md
**Purpose:** Analyze, plan, and execute Vercel SDK upgrades with breaking change detection and rollback procedures.
**Workflow:** Used during SDK version upgrades - typically quarterly or when new features are needed.
**Relates to:** Follows vercel-prod-checklist for deployment; CI integration via vercel-ci-integration

## Summary

This skill guides developers through safe SDK version upgrades. It covers checking current versions, reviewing changelogs for breaking changes, creating upgrade branches, handling import and configuration changes between versions, running test suites, and documenting rollback procedures. The deprecation handling section shows how to detect and address deprecated APIs before they're removed.
