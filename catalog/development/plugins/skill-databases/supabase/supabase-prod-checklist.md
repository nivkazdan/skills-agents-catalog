## Summary

This skill provides a comprehensive production deployment checklist for Supabase integrations. It covers pre-deployment configuration, code quality verification, infrastructure setup, documentation requirements, and gradual rollout with canary deployment. Use this skill when preparing to deploy a Supabase integration to production or when conducting pre-launch reviews.
