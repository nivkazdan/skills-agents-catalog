## File Descriptions

### 1. SKILL.md
**Purpose:** Diagnose and fix the top 10 most common Supabase errors including Invalid JWT, RLS Policy Violation, and Connection Pool Exhausted.
**Workflow:** First troubleshooting skill. Used when encountering errors during development or production.
**Relates to:** Entry point for troubleshooting; escalates to supabase-debug-bundle for complex issues.

## Summary

This skill is a quick reference for diagnosing and fixing common Supabase errors. It covers Invalid JWT errors, RLS policy violations, connection pool exhaustion, and other frequently encountered issues. Use this skill when you encounter a Supabase error and need a fast resolution. Each error includes the error message, cause, and step-by-step solution.
