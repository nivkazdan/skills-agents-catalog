## File Descriptions

### 1. SKILL.md
**Purpose:** Configure Supabase local development with hot reload and testing workflows. Establishes a fast iteration cycle for Supabase development.
**Workflow:** Third skill in the onboarding sequence. Sets up the development environment that will be used throughout the project lifecycle.
**Relates to:** Follows supabase-hello-world; enables efficient development for all subsequent skills.

## Summary

This skill establishes a productive local development workflow for Supabase projects. It covers project structure setup, environment configuration, hot reload with tsx/nodemon, test suite configuration with Vitest, and mocking Supabase responses. Use this skill when you need to set up a new development environment or standardize development practices across a team.
