## File Descriptions

### 1. SKILL.md
**Purpose:** Execute Supabase incident response procedures with severity levels, quick triage, decision trees, and postmortem templates.
**Workflow:** Incident response skill. Used during Supabase-related outages or degradations.
**Relates to:** Depends on supabase-observability for monitoring data; uses supabase-debug-bundle for evidence collection.

## Summary

This skill provides a rapid incident response framework for Supabase-related outages. It covers severity level definitions, quick triage commands, decision tree for identifying issue source, immediate actions by error type (401/403, 429, 500/503), communication templates for internal and external stakeholders, and postmortem template. Use this skill when responding to Supabase incidents or when setting up on-call procedures.
