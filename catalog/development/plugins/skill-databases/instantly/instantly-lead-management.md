# instantly-lead-management

```
instantly-lead-management/
```

## File Descriptions

| File | Type | Purpose |
|------|------|---------|
| SKILL.md | Markdown | Lead management workflow and best practices |
| import_leads.py | Python | Bulk import leads with validation and error handling |
| deduplicate_leads.py | Python | Remove duplicate leads across campaigns |

| validate_emails.py | Python | Verify email validity before import |
