# posthog-funnels-analysis

> Build and analyze conversion funnels to understand user journey and identify drop-off points

## Directory Structure

```
posthog-funnels-analysis/
├── SKILL.md
└── examples/
    ├── funnel_definition.json
    ├── conversion_analysis.md
    ├── funnel_api.py
    └── optimization_playbook.md
```

## File Descriptions

| File | Type | Purpose |
|------|------|---------|
| SKILL.md | Markdown | Core skill instructions for building and analyzing PostHog funnels |
| funnel_definition.json | JSON | Example funnel configuration with steps and filters |
| conversion_analysis.md | Markdown | Guide for interpreting funnel metrics and drop-off data |
| funnel_api.py | Python | Script to query funnel data via PostHog API |
| optimization_playbook.md | Markdown | Playbook for funnel optimization based on analysis |
