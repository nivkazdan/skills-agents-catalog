## File Descriptions

### 1. SKILL.md
**Purpose:** Migrate to Sentry from other error tracking tools like Rollbar, Bugsnag, Raygun, or NewRelic with SDK replacement guides and feature mapping.
**Workflow:** Use when switching from another error tracking tool. Provides complete migration path with parallel running and validation.
**Relates to:** Works with `sentry-install-auth` for new setup. Connects to `sentry-common-errors` for post-migration issues.

## Summary

This skill provides a comprehensive guide to migrating from other error tracking tools to Sentry. It covers migration planning phases, SDK replacement for Rollbar, Bugsnag, Raygun, and NewRelic, feature mapping between tools, data migration options, alert migration, testing migration, parallel running, cleanup, and dependency removal. Use this skill when switching error tracking providers.
