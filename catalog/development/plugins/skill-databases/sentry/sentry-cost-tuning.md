## File Descriptions

### 1. SKILL.md
**Purpose:** Optimize Sentry costs and event volume through sampling, filtering, and disabling unused features while maintaining monitoring quality.
**Workflow:** Apply when managing Sentry billing or proactively optimizing spend. Implements cost-reduction strategies across the organization.
**Relates to:** Works with `sentry-rate-limits` for quota management. Connects to `sentry-load-scale` for high-volume optimization.

## Summary

This skill provides strategies to optimize Sentry costs while maintaining visibility. It covers understanding Sentry pricing, reducing error volume through filtering, reducing transaction volume through sampling, optimizing event size, disabling unused features, cost monitoring via API, tiered environment configuration, and monthly cost estimation. Use this skill when managing Sentry spending.
