## Summary

**Category:** enterprise
**Target Audience:** Compliance officer
**Trigger Phrases:** `klingai compliance`, `kling ai audit`, `klingai security review`, `klingai gdpr`

### What This Skill Does

This skill conducts compliance reviews for Kling AI integrations. It covers:

- Compliance frameworks (GDPR, SOC2, HIPAA)
- Security assessment checklists
- Data handling review
- Access control verification
- Gap analysis and scoring
- Remediation planning
- Audit preparation

### Technical Success Criteria

- Documented compliance status with remediation plan
- Gap analysis completed
- Evidence collection for auditors

### Business Success Criteria

- Audit-ready compliance posture
- Clear roadmap to full compliance
- Stakeholder confidence in compliance
