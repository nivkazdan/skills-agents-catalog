## File Descriptions

### 1. SKILL.md
**Purpose:** Diagnose and fix Clay common errors including API rate limits, enrichment failures, authentication issues, and integration problems. Provides error-specific troubleshooting flows.
**Workflow:** Troubleshooting skill. Use when encountering Clay errors that need rapid resolution.
**Relates to:** Supports all Clay skills; often leads to clay-debug-bundle for complex issues.

## Summary

This skill provides a comprehensive guide to diagnosing and fixing common Clay errors. It covers API authentication failures (401/403), rate limit errors (429), enrichment failures (missing data, provider errors), webhook delivery issues, and CRM sync problems. Each error type includes identification patterns, root cause analysis, and step-by-step resolution procedures.
