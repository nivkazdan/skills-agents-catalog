## File Descriptions

### 1. SKILL.md
**Purpose:** Implement Clay PII handling, data retention, and GDPR/CCPA compliance patterns. Ensures proper handling of enriched prospect data.
**Workflow:** Compliance skill. Use during security audits and compliance implementations.
**Relates to:** Builds on clay-security-basics; integrates with clay-enterprise-rbac for access control.

## Summary

This skill implements data handling compliance for Clay integrations. It covers classifying enriched data (emails, phone numbers, company info), implementing PII detection and redaction in logs, configuring data retention policies, handling GDPR data subject requests, implementing CCPA compliance for California residents, and creating audit trails for compliance reporting. Essential for regulated industries.
