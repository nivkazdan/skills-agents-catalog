### 1. SKILL.md
**Purpose:** Implement Clay reference architecture with best-practice project layout. Provides standardized patterns for team alignment and maintainability.
**Workflow:** Architecture skill. Use when starting new Clay projects or refactoring existing ones.
**Relates to:** Synthesizes patterns from all Clay skills; foundation for clay-multi-env-setup.

## Summary

This skill implements a reference architecture for Clay integrations. It covers structured project layout, API client wrapper with caching and retry logic, error boundary implementation, health check endpoints, configuration management patterns, and logging/monitoring integration. Provides a proven starting point for new Clay projects with team-aligned patterns.
