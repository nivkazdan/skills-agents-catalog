## File Descriptions

### 1. SKILL.md
**Purpose:** Install and configure Clay API authentication. Sets up API key configuration, environment variables, and verifies connectivity to Clay enrichment services.
**Workflow:** Entry point skill for any Clay integration. Should be the first skill executed when setting up a new project with Clay.
**Relates to:** Prerequisite for clay-hello-world, clay-local-dev-loop, and all other Clay skills.

## Summary

This skill guides developers through the complete Clay API installation and authentication setup. It covers installing HTTP client libraries, configuring API keys through environment variables or .env files, and verifying the connection works correctly with a simple API health check. Use this skill when starting a new Clay integration or when onboarding new team members to an existing Clay project.
