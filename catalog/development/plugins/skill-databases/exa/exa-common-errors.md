## File Descriptions

### 1. SKILL.md
**Purpose:** Diagnose and fix top 10 most common Exa errors including rate limits, invalid queries, authentication failures, and content extraction issues.
**Workflow:** First-line troubleshooting resource - covers 90% of issues developers encounter.
**Relates to:** Follows any onboarding skill; precedes exa-debug-bundle for complex issues

## Summary

This skill documents the 10 most common Exa errors with diagnosis and resolution steps: 401 Authentication errors (invalid/missing API key), 429 Rate limit exceeded, 400 Invalid query format, empty results (query optimization), content extraction failures, timeout errors, invalid domain filters, date range errors, SDK version mismatches, and network connectivity issues. Each error includes symptoms, root cause, and step-by-step resolution.
