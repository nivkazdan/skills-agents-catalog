## File Descriptions

### 1. SKILL.md
**Purpose:** Analyze, plan, and execute Exa SDK upgrades with breaking change detection and rollback procedures.
**Workflow:** Maintenance skill for keeping SDK current - includes migration automation.
**Relates to:** Follows exa-sdk-patterns; applies to all Exa codebases

## Summary

This skill manages Exa SDK upgrades: checking current vs latest versions, reviewing changelog for breaking changes, analyzing codebase for affected patterns, creating upgrade branch with changes, running test suite to verify compatibility, documenting rollback procedure, and executing phased rollout. Includes scripts for automated upgrade analysis and common migration patterns between versions.
