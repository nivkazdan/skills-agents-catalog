## File Descriptions

### 1. SKILL.md
**Purpose:** Implement domain filtering, date ranges, and category filters for precise search targeting and result quality.
**Workflow:** Precision search skill - essential for focused research and curated content retrieval.
**Relates to:** Follows exa-search-modes; enhances exa-content-extraction and exa-similarity-search

## Summary

This skill covers Exa's powerful filtering capabilities: includeDomains and excludeDomains for source control, startPublishedDate and endPublishedDate for temporal filtering, and category filters for content type targeting. Includes patterns for building filter presets (e.g., "trusted news sources only"), combining multiple filters effectively, and dynamic filter generation based on user preferences or use case requirements.
