## Usage

Skills trigger automatically when you discuss Replit topics. For example:

- "Help me set up Replit" → triggers `replit-install-auth`
- "Debug this Replit error" → triggers `replit-common-errors`
- "Deploy my Replit integration" → triggers `replit-deploy-integration`
