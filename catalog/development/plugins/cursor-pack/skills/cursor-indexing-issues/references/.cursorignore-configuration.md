# .Cursorignore Configuration

## .cursorignore Configuration

### Basic Setup
```gitignore
# .cursorignore (at project root)

# Dependencies
node_modules/
vendor/
.venv/
__pycache__/

# Build outputs
dist/
build/
out/
.next/
.nuxt/

# Version control
.git/

# Large files
*.log
*.csv
*.sql
*.sqlite
*.db

# Data directories
data/
datasets/
uploads/

# IDE/Editor
.idea/
.vscode/
*.swp
```

### Advanced Patterns
```gitignore
# Ignore all but include specific
*
!src/
!lib/
!*.py
!*.ts

# Size-based exclusion (manual)
# Files > 1MB should be listed explicitly
large-dataset.json
```
