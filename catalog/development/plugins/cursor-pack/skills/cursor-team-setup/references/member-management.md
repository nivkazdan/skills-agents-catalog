# Member Management

## Member Management

### Inviting Members
```
Admin Dashboard > Members > Invite

Options:
- Email invitation
- Domain-based auto-join
- SSO provisioning (Enterprise)

Invite by email:
1. Enter email addresses
2. Select role (Admin/Member)
3. Send invitations
4. Track pending invites
```

### Role Management
```
Roles:

Owner:
- Full account control
- Billing management
- Delete organization

Admin:
- Manage members
- Configure settings
- View analytics
- Cannot delete org

Member:
- Use all features
- No admin access
- Cannot invite users
```

### Removing Members
```
Admin Dashboard > Members > Select User > Remove

Considerations:
- Immediate access revocation
- Settings preserved for 30 days
- Can be reinvited later
- Billing adjusts automatically
```
