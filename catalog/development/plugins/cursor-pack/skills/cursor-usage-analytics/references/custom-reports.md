# Custom Reports

## Custom Reports

### Creating Reports
```
Admin > Analytics > Custom Reports

Report Types:
- Usage summary (daily/weekly/monthly)
- User activity
- Cost breakdown
- Feature adoption
- Team comparison
```

### Scheduled Reports
```
Automate reporting:

1. Create report template
2. Set schedule (daily/weekly/monthly)
3. Add recipients
4. Choose format (PDF/CSV/Email)
5. Enable delivery

Example schedules:
- Weekly usage summary to managers
- Monthly cost report to finance
- Daily activity to team leads
```

### Export Options
```
Export formats:
- CSV (raw data)
- PDF (formatted report)
- JSON (API integration)

Export via:
- Dashboard download
- Scheduled export
- API endpoint
```
