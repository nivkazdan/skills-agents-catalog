# Scripts

Bundled resources for docker-compose-generator skill

- [x] validate_compose.sh: Validates the generated Docker Compose file using docker-compose config.
- [x] generate_env_file.py: Generates a .env file based on the Docker Compose file, pre-filled with default values.
- [x] deploy.sh: A script to deploy the Docker Compose file to a Docker Swarm or Kubernetes cluster.
