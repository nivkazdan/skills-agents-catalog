## Usage

Skills trigger automatically when you discuss Instantly topics. For example:

- "Help me set up Instantly" → triggers `instantly-install-auth`
- "Debug this Instantly error" → triggers `instantly-common-errors`
- "Deploy my Instantly integration" → triggers `instantly-deploy-integration`
