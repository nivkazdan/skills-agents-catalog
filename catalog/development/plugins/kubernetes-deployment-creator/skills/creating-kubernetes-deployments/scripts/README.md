# Scripts

Bundled resources for kubernetes-deployment-creator skill

- [x] validate_manifest.sh: Validates a Kubernetes manifest file against the Kubernetes API schema.
- [x] apply_manifest.sh: Applies a Kubernetes manifest file to the cluster.
- [x] delete_manifest.sh: Deletes a Kubernetes manifest file from the cluster.
- [x] get_resource_status.sh: Retrieves the status of a Kubernetes resource (e.g., deployment, service).
