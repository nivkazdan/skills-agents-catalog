## Usage

Skills trigger automatically when you discuss CodeRabbit topics. For example:

- "Help me set up CodeRabbit" → triggers `coderabbit-install-auth`
- "Debug this CodeRabbit error" → triggers `coderabbit-common-errors`
- "Deploy my CodeRabbit integration" → triggers `coderabbit-deploy-integration`
