## Usage

Skills trigger automatically when you discuss Groq topics. For example:

- "Help me set up Groq" → triggers `groq-install-auth`
- "Debug this Groq error" → triggers `groq-common-errors`
- "Deploy my Groq integration" → triggers `groq-deploy-integration`
