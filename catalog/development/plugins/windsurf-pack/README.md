## Usage

Skills trigger automatically when you discuss Windsurf topics. For example:

- "Help me set up Windsurf" → triggers `windsurf-install-auth`
- "Debug this Windsurf error" → triggers `windsurf-common-errors`
- "Deploy my Windsurf integration" → triggers `windsurf-deploy-integration`
