## Usage

Skills trigger automatically when you discuss Fireflies.ai topics. For example:

- "Help me set up Fireflies.ai" → triggers `fireflies-install-auth`
- "Debug this Fireflies.ai error" → triggers `fireflies-common-errors`
- "Deploy my Fireflies.ai integration" → triggers `fireflies-deploy-integration`
