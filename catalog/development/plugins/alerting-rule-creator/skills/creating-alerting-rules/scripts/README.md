# Scripts

Bundled resources for alerting-rule-creator skill

- [x] validate_thresholds.py: Validates alerting thresholds against historical data to prevent false positives.
- [x] route_alert.py: Routes alerts based on severity and category to appropriate channels (e.g., Slack, PagerDuty).
- [x] generate_runbook.py: Generates a basic runbook based on the alert type and potential causes.
